# Modern C++ TUI Template (Ncursesw + Panels)

Профессиональный шаблон для создания текстовых пользовательских интерфейсов (**TUI**) в терминале Linux. Написан на современном C++ (стандарт C++17) с использованием библиотеки **Ncursesw**, менеджера слоев **Panels**, а также модулей форм и меню.

## ✨ Особенности проекта
* **Полная поддержка UTF-8 (Кириллицы):** Решена классическая проблема "кракозябр" Ncurses в Linux (ввод и вывод на русском языке работают стабильно).
* **Модульная архитектура:** Код разделен на независимые логические компоненты (`main`, `popup`, `settings`, `dataview`).
* **Многослойный интерфейс (Z-Order):** Всплывающие модальные окна реализованы через `panel.h` и не затирают задний фон при закрытии.
* **Кастомная система ввода:** Стандартный баг формы при переключении по `Tab` исправлен с помощью ручной фильтрации очереди сообщений через `wget_wch`.
* **Поддержка мыши:** Пункты меню и фокус полей ввода реагируют на клики мыши как в графических эмуляторах, так и в "чистом" TTY-терминале.
* **Событийно-ориентированный рендеринг:** Экран перерисовывается только при изменении состояния, что исключает мерцание интерфейса.

## 🛠️ Системные требования и зависимости

Проект протестирован и гарантированно собирается на **Debian 13**.

Для сборки вам понадобятся компилятор с поддержкой C++17, CMake, инструмент `pkg-config` и пакеты разработки Ncurses (включающие широкую UTF-8 версию библиотеки):

```bash
sudo apt update
sudo apt install build-essential cmake pkg-config libncurses-dev
```

### Поддержка мыши в чистом TTY (без графики)
Если вы запускаете проект в чистой консоли Linux (`Ctrl + Alt + F1`), установите фоновую службу GPM для трансляции событий мыши:
```bash
sudo apt install gpm
```

## 🚀 Сборка и запуск

Сборка проекта выполняется стандартным для CMake способом:

```bash
# 1. Создаем директорию сборки и переходим в нее
mkdir build && cd build

# 2. Генерируем конфигурационные файлы сборки
cmake ..

# 3. Компилируем исполняемый файл
cmake --build .

# 4. Запускаем приложение
./test
```

## 📁 Структура исходного кода

* `CMakeLists.txt` — Скрипт сборки с жесткой и безопасной раздельной линковкой модулей `panelw`, `menuw`, `formw` и `ncursesw`.
* `main.cpp` — Точка входа, инициализация цветовой палитры и логика Главного меню.
* `popup.hpp` / `popup.cpp` — Модуль для динамического создания и удаления цветных всплывающих панелей.
* `settings.hpp` / `settings.cpp` — Форма настроек с многоколоночным вводом русских слов на чистом Ncurses.
* `dataview.hpp` / `dataview.cpp` — Прокручиваемый список (Scrollable List) для вывода табличных данных с привязкой объектов через `userptr`.

## ⌨️ Управление
* **Стрелки Вверх / Вниз:** Навигация по меню и спискам данных.
* **Tab / Shift+Tab:** Переключение между текстовыми полями в настройках.
* **Enter:** Выбор пункта меню / Сохранение формы.
* **Левый клик мыши:** Клик по меню или выбор поля ввода.
* **Q / Esc:** Выход из текущего всплывающего окна.
